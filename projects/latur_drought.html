Drought Assessment in Northern Latur (2010-2022)

Remote Sensing Shivaji University Guide: Dr. Sudhir K. Powar

Project Objective

This research provided an integrated approach using remote sensing and meteorological data to map and assess drought vulnerability across the Northern Latur district over a 12-year temporal period.

Figure 1: Automated outputs of geospatial indices (NDVI, NDWI, NDMI) processed from Landsat imagery.

Technical Approach

  • Python Automation: Developed custom Python scripts to automate the processing and map generation of temporal Landsat imagery, significantly reducing manual raster math.
  • Index Calculation: Automated the extraction of critical environmental indices including NDVI (Vegetation), NDWI (Water), and NDMI (Moisture).
  • Meteorological Integration: Utilized the Standardized Precipitation Index (SPI-3) to characterize precipitation deficits across administrative boundaries.
  • CDHI Modeling: Built a Comprehensive Drought Hazard Index (CDHI) model using weighted overlay spatial analysis to accurately classify varying levels of drought severity.
